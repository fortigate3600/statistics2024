Teorema Fondamentale del Calcolo

Il Teorema Fondamentale del Calcolo collega il concetto di derivata e quello di integrale, fornendo una base teorica per il calcolo integrale. Esso si compone di due parti principali: la prima parte afferma che l'integrale definito di una funzione continua 𝑓 ( 𝑥 ) f(x) su un intervallo [ 𝑎 , 𝑏 ] [a,b] può essere valutato utilizzando una primitiva 𝐹 ( 𝑥 ) F(x) di 𝑓 ( 𝑥 ) f(x), tale che ∫ 𝑎 𝑏 𝑓 ( 𝑥 )   𝑑 𝑥 = 𝐹 ( 𝑏 ) − 𝐹 ( 𝑎 ) ∫ a b ​ f(x)dx=F(b)−F(a). La seconda parte stabilisce che, se 𝐹 ( 𝑥 ) F(x) è definita come l'integrale della funzione 𝑓 ( 𝑥 ) f(x) a partire da un punto fisso 𝑎 a, allora la derivata di 𝐹 ( 𝑥 ) F(x) rispetto a 𝑥 x è uguale a 𝑓 ( 𝑥 ) f(x), ovvero 𝐹 ′ ( 𝑥 ) = 𝑓 ( 𝑥 ) F ′ (x)=f(x). Questo teorema è fondamentale anche nella teoria della probabilità, dove si collega alla funzione di densità di probabilità (PDF) e alla funzione di distribuzione cumulativa (CDF).

In particolare, data una variabile casuale continua 𝑋 X con funzione di densità di probabilità 𝑓 ( 𝑥 ) f(x), la funzione di distribuzione cumulativa 𝐹 ( 𝑥 ) F(x) rappresenta la probabilità che 𝑋 X assuma un valore minore o uguale a 𝑥 x, ovvero 𝐹 ( 𝑥 ) = 𝑃 ( 𝑋 ≤ 𝑥 ) = ∫ − ∞ 𝑥 𝑓 ( 𝑡 )   𝑑 𝑡 F(x)=P(X≤x)=∫ −∞ x ​ f(t)dt. In questo contesto, il Teorema Fondamentale del Calcolo implica che la derivata della CDF 𝐹 ( 𝑥 ) F(x) rispetto a 𝑥 x è uguale alla PDF 𝑓 ( 𝑥 ) f(x), quindi 𝐹 ′ ( 𝑥 ) = 𝑓 ( 𝑥 ) F ′ (x)=f(x). Pertanto, il teorema fornisce una relazione tra la CDF e la PDF di una variabile casuale continua, dimostrando come la PDF rappresenti la "densità" locale di probabilità e la CDF accumuli questa densità lungo l'intervallo dei valori di 𝑋 X.
